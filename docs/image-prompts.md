# UI確認用 仮画像プロンプト

現在は `src/assets/images/generated/` に、UI確認用の仮画像を同じファイル名・同じ比率で配置しています。公開前に実写、または改めて生成した画像へ差し替えてください。画像内に文字、ロゴ、看板文字、読める商品ラベルは入れません。

## 共通方針

- 愛媛県松山市・中島をイメージした瀬戸内の静かな島の風景
- 観光広告ではなく、生活のドキュメンタリー
- 自然光、少し曇った柔らかい光
- 彩度は控えめ
- 落ち着いた生成り、海の青灰、木の茶、レモンの黄色
- フィルム写真のようなやわらかな空気感
- 人物はカメラ目線にしない
- 生活や作業の痕跡が見える
- Webサイト上に文字を重ねやすい余白を確保する
- 過度に美化しすぎず、リアルで静かな雰囲気にする
